# VolcanoUI

### Install

<hr/>

```
npm i volcano-ui -S
```

### Quick Start

<hr/>

In main.js:

```
import Vue from 'vue';
import VolcanoUI from 'volcano-ui';
import 'bee-ui/lib/volcano-ui.css';
import App from './App.vue';
Vue.use(VolcanoUI);
```
