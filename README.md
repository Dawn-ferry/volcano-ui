### Install

<hr/>

```
npm i volcano-ui -S
```

### Quick Start

<hr/>

In main.js:

```
import Vue from 'vue';
import App from './App.vue';
import VolcanoUI from 'volcano-ui'
import 'volcano-ui/lib/volcano-ui.css'
Vue.use(VolcanoUI);

```
